# Predictive Modeling Using Machine Learning

## 📌 Project Overview
This project uses a Linear Regression model to predict house prices based on the area of the house. It demonstrates the fundamentals of supervised machine learning, including model training, testing, evaluation, and visualization.

## 🚀 Features
- Data preprocessing using Pandas
- Train-Test Split for model evaluation
- Linear Regression model implementation
- House price prediction
- Performance evaluation using R² Score and Mean Squared Error
- Graph visualization using Matplotlib

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib

## 📂 Dataset

| Area (sq ft) | Price (Lakhs) |
|---------------|---------------|
| 1000 | 30 |
| 1200 | 35 |
| 1500 | 45 |
| 1800 | 50 |
| 2000 | 60 |
| 2200 | 65 |
| 2500 | 70 |
| 2800 | 80 |
| 3000 | 85 |
| 3500 | 95 |

## 📊 Model Evaluation
- R² Score calculated for accuracy measurement
- Mean Squared Error (MSE) used for performance evaluation

## 📈 Visualization
The project displays a regression graph showing:
- Actual house prices
- Predicted regression line

## ▶️ How to Run

1. Install required libraries:

```bash
py -m pip install pandas scikit-learn matplotlib
```

2. Run the project:

```bash
py main.py
```

## 📷 Output
- Predicted house price based on input area
- R² Score and Mean Squared Error
- Regression graph visualization
